# Fotographer

Creative tools for image art and design. High-performance, real-time effects powered by pure TypeScript and Canvas 2D.

## Features

- **10 Unique Effects**: From retro halftones to psychedelic Droste zooms.
- **Real-time Preview**: Debounced canvas processing for smooth parameter adjustment.
- **Gallery**: Save your creations to local storage.
- **Share Links**: Reproduce exact effects by sharing encoded URLs.
- **Pure TypeScript**: No external image processing libraries used.
- **Responsive Design**: Optimized for both desktop and mobile users.

## Effects List

1. **ToneKit**: Retro halftone dot-matrix.
2. **BlurSuite**: Creative blur modes (Linear, Radial, Zoom, Wave).
3. **Super-G**: Glitch art with slicing and color shifts.
4. **Scanline**: CRT/VHS analog corruption.
5. **ReColor**: Hue remapping and color grading.
6. **ASCIIKit**: Image to ASCII character art.
7. **Retroman**: Pixel dithering and retro palettes.
8. **Glassify**: Glass refraction distortion.
9. **ImageTrack**: Artistic generative blob tracking.
10. **LoopFlow**: Droste infinite zoom effect.

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
